# VJA - Python pentru incepatori

Curs gratuit de Python pentru copii.

## Structura proiectului

```
docs/           - materialele pentru fiecare lectie
  lectia-1.md   - Lectia 1: Introducere
  instalare-python.md - Ghid de instalare Python + VSCode
src/            - fisierele cu cod Python
  showcase.py   - Mini-joc "Prinde Cercurile" (demo pentru Lectia 1)
```

## Cum incep?

### 1. Instalare

Urmeaza ghidul pas cu pas din [docs/instalare-python.md](docs/instalare-python.md).

Pe scurt:
- Instaleaza **Python** de pe https://www.python.org/downloads/ (bifeaza "Add to PATH"!)
- Instaleaza **VSCode** de pe https://code.visualstudio.com/
- Instaleaza extensia **Python** in VSCode (Ctrl + Shift + X, cauta "Python")

### 2. Rulare script

Deschide un fisier `.py` in VSCode si apasa butonul **â–¶ (Play)** din dreapta sus.

Sau din terminal:

```bash
python src/showcase.py
```

Pe Mac/Linux poate fi nevoie de `python3` in loc de `python`.

### 3. Alternativa online

Daca nu poti instala Python, foloseste: https://www.pythonsandbox.com
